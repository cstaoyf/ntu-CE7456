\extraspacing {\bf Problem 1 (10 marks).} Set $S = \set{e_1, e_2, ..., e_5}$ where each element carries a positive integer weight. The following describes an alias structure on $S$:
\myitems{
    \item Urn 1: elements $e_1$ and $e_2$, associated with values $10/5$ and $2/5$, respectively;
    \item Urn 2: elements $e_2$ and $e_4$, associated with values $8/5$ and $4/5$, respectively;
    \item Urn 3: element $e_3$, associated with value $12/5$;
    \item Urn 4: elements $e_1$ and $e_5$, associated with values $1/5$ and $11/5$, respectively;
    \item Urn 5: elements $e_2$ and $e_5$, associated with values $9/5$ and $x/5$, respectively.
}
Answer the following questions:
\myenums{
    \item What is the value of $x$ (note: $x/5$ is the value of $e_5$ in Urn 5)?
    \item If we perform weighted sampling on $S$, what is the probability that $e_2$ is sampled?
    \item To draw a weighted sample from $S$, the algorithm of the alias structure first chooses an urn uniformly at random. Suppose that Urn 4 has been chosen. Conditioned on this fact, what is the probability for $e_5$ to be returned?
}

\extraspacing {\bf Problem 2 (10 marks).} Consider the IQS version of the 1D range reporting problem discussed in our lecture notes. Recall that, given an interval $q = [x, y]$, an IQS operation on a set $S$ of $n$ integers --- where each integer carries a positive weight --- performs weighted sampling on $q \cap S$. Decide whether each of the following statements is true; if you believe a statement is false, you need to provide a justification.
\myenums{
    \item If we issue two IQS operations using the same interval $q$, their outputs must be independent from each other.
    \item If we issue an infinite number of IQS operations with the interval $q = (-\infty, \infty)$, we can definitely find all the elements of $q \cap S$.
    \item Suppose that all the integers in $S$ carry the same weight. Given any interval $q$, we can perform an IQS operation in $O(\log n)$ time using only an array where the elements of $S$ are stored in ascending order.
}

\extraspacing {\bf Problem 3 (10 marks).} Let $P$ be a set of $n$ points in $\real^2$. Given an axis-parallel rectangle $q$, a {\em counting} query returns $|P \cap q|$, i.e., how many points of $P$ are covered by $q$. It is known that, in $O(n \log n)$ time, we can construct a {\em counting tree} that uses $O(n \log n)$ space and answers a counting query in $O(\log n)$ time. Answer the following questions.

\myenums{
    \item Prove: There is a semi-dynamic structure of $O(n \log n)$ space that can answer a counting query on $P$ in $O(\log^2 n)$ time and can support insertions into $P$ in $O(\log^2 n)$ amortized time.

    \item Consider the {\em batched counting problem} defined as follows. The input includes a set $P$ of $n$ points in $\real^2$ and a set $Q$ of $m$ axis-parallel rectangles where $m \le n$. Suppose that both $P$ and $Q$ are given as arrays, with elements stored in arbitrary order. The goal is to output the size $|P \cap q|$ for each $q \in Q$. Explain how to solve the problem in $O(n \log m)$ time.

    \vgap

    Hint: deferred data structuring.
}

\extraspacing {\bf Problem 10 (10 marks).} Let $\Q$ be a join whose schema graph is $\H = (\V, \E)$. Set $d = |\V|$, and denote by $\In$ the input size of $\Q$. Let $\W$ be an arbitrary optimal fractional edge cover of $\H$. We know that
\myitems{
    \item $\In$ is a power of 2.

    \item Attribute values lie in the so-called {\em rank space}. Specifically, for any relation $R \in \Q$, any attribute $A \in \schema(R)$, and any tuple $\bm{u} \in R$, the value $\bm{u}(A)$ is an integer in $[\In]$.
}
In the lecture notes, we created a box tree $\T$ as follows. The root of $\T$ is associated with the box $[\In]^d$. In general, consider a node associated with box $B$. If $B$ has an AGM value under $\W$ at most $1$, we make $B$ a leaf of $\T$. Otherwise, we split $B$ evenly along some arbitrary dimension into boxes $B_1$ and $B_2$. We then create two child nodes of $B$, associated with $B_1$ and $B_2$, respectively. We proved that $\T$ has $O(\agm \cdot \log \In)$ nodes, where $\agm$ is the AGM bound of $\Q$.

\vgap

Now, given an integer $t \ge 1$, we construct $\T$ in slightly different manner. The root of $\T$ is associated with the box $[\In]^d$. In general, consider a node associated with box $B$. If $B$ has an AGM value under $\W$ \uline{at most $t$}, we make $B$ a leaf of $\T$. Otherwise, we split $B$ evenly along some arbitrary dimension into boxes $B_1$ and $B_2$. We then create two child nodes of $B$, associated with $B_1$ and $B_2$, respectively.

\vgap

Prove: The new $\T$ has $O(\fr{\agm}{t} \cdot \log \In)$ nodes.
